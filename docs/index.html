Quick Start Guide

1. Authenticate GitHub CLI Ensure you are logged in: gh auth login
2. Set your API Key Export your Gemini key or add it to a .env file: export GEMINI_API_KEY=your_key
3. Start the Transmutation Optimize your profile or repo metadata: python3 -m src.cli topics --smart
